# Martin Tasev Portfolio

Modern developer portfolio built with Next.js, Tailwind CSS and Framer Motion.
The website is designed as a freelance presentation platform for showcasing projects, technical skills and enabling direct collaboration through a working contact system.

---

## Overview

This project represents a fully animated single-page portfolio focused on performance, minimal design and smooth user experience.
It includes theme switching, scroll-based UI interactions, project presentation slider and email contact integration.

---

## Tech Stack

* Next.js (App Router)
* Tailwind CSS
* Framer Motion
* Resend Email API

---

## Features

* Dark / Light theme toggle
* Animated radial gradient background
* Scroll progress indicator
* Responsive layout (mobile friendly)
* Project slider with swipe support on mobile
* Animated section transitions
* Functional contact form with email delivery
* Social links navigation
